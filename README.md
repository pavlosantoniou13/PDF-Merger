Adobe != Free so i created this to merge pdfs.

How to use: 

1. Prerequisites
You’ll need Python installed. This app uses Streamlit for the UI, pypdf for merging, and ReportLab for generating the divider pages.

2. Installation
Clone this repo and install the requirements:
git clone https://github.com/your-username/pdf-divider-merger.git
cd pdf-divider-merger
pip install streamlit pypdf reportlab

3. Run the App
Bash
streamlit run app.py
